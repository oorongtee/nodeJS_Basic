# nodeJS基礎練習

> 本專案練習由node.js 來寫後端的API資料，做了CRUD的API操作

> 採用MVC架構
 
> 感謝鐵人賽>>2魚的手把手教學：https://ithelp.ithome.com.tw/m/users/20159686/ironman/7564

# 專案技術
- node.js
- Express
- MongoDB / mongoose
- Swagger
- cors
- dotenv
- uuid

# 功能
- 連接MongoDB，並實現CRUD
- Swagger紀錄API並輸出文件
- API測試：Postman是個好東西
- API資料模型，路由與測試
- 可重用函式設計與全面錯誤管理
- gitignore可防止檔案上傳到git上
